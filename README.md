# 实验室失物招领微信小程序

## 功能概览
- 老师录入：拍照识别分类（模拟识别）、手动补充物品信息。
- 学生查询：首页按分类/实验区/时间筛选，查看详情并提交认领申请（含证件照与理由）。
- 老师审核：通过/拒绝认领申请，通过后记录“远程开锁”。
- 历史追溯：认领记录、物品流转记录、CSV 报表导出（可用 Excel 打开）。
- 扩展功能：学生发布、管理寻物启示（标记已找到/删除）。

## 目录
- `pages/index`：学生首页 + 筛选
- `pages/item-detail`：失物详情 + 认领
- `pages/teacher-add`：老师录入
- `pages/teacher-review`：老师审核
- `pages/records`：历史记录与导出
- `pages/bulletin`：寻物启示
- `utils/store.js`：本地存储与核心数据逻辑

## 使用方式
1. 用微信开发者工具打开项目目录。
2. 进入首页切换角色（学生/老师）并体验各模块。
3. 报表导出在“历史记录”页，生成路径为小程序用户目录下的 `lost-found-report.csv`。

## 说明
- 当前“拍照识别”为基于文件名关键词的本地模拟识别，方便离线演示。
- 智能锁开锁流程为业务状态模拟，实际落地可对接校园物联网 API。

## 打包下载
- 运行：`python scripts/package_zip.py`
- 默认输出：`/workspace/lost-found-miniapp.zip`
- 打包时会自动排除 `.git`、`.vscode`、`node_modules` 等非发布文件。
